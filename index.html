Interludes

A full stack MERN app, with state management using React Context, routing using React Router. Mongoose is used with MongoDB Atlas to store the backend data. Check the design and demo for a quick overview, and use user: test@test.com and password: test1234 to give it a spin.

NEO Today

View the five closest near-earth objects passing by our planet today. Featuring data from NASA Open API, an interactive UI using jQuery, and a display that features animated 3d objects usingThreeJS.

Local Scene

An event calendar for the local music scene using an animated neomorphism design. Utililizes PHP for dynamic page views and phpMyAdmin on cPanel as the database.

Gin Scorer

A React app which uses a Firestore database and authentication using Google OAuth through the Firebase Javascript SDK. The app tracks the score of a game of Gin, as well as the overall win rate of a pair of players. Features custom animations created in Adobe After Effects using the LottiFiles library.

Introspect

An app utilizing jQuery to create a custom meditation card based on user input, which is saved locally using the Web Storage API.
